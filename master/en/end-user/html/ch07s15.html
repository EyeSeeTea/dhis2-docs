7.15.1. About legends

You can create your own legend to make sure that maps created in the GIS app are meaningful.

Note

Continuous legends are supposed to end and start on the same value, for example 0-50 and 50-80. The GIS app automatically take care of this. Do not try to do this yourself by setting legends to for example 0-50 and 51-80. This will cause a usually unwanted gap in your legend set.

In the Maintenance app, you manage the following legend objects:

7.15.2. Create a legend

Note

It is not allowed to have gaps in a legend.

It is not allowed to have overlapping legend items.

  1. Open the Maintenance app and click Other > Legend.

  2. Click the add button.

  3. In the Name field, type the legend name.

  4. (Optional) In the Code field, assign a code.

  5. Create the legend items you want to have in your legend:

    1. Select Start value and End value.

    2. Select Number of items.

    3. Select a color scheme.

    4. Click Create legend items.

    Tip

    Click the options menu to edit or delete a legend item.

  6. (Optional) Add more legend items:

    1. Click the add button

    2. Enter a name and select a start value, an end value and a color.

    3. Click OK.

  7. Click Save.

7.15.3. Edit legends

  1. Open the Maintenance app and click Other > Legend.

  2. Click Legend.

  3. In the list, find the legend you want to modify.

  4. Click the options menu and select Edit.

  5. Modify the options you want.

  6. Click Save.

7.15.4. Clone legends

  1. Open the Maintenance app and click Other > Legend.

  2. Click Legend.

  3. In the list, find the legend you want to clone.

  4. Click the options menu and select Clone.

  5. Modify the options you want.

  6. Click Save.

7.15.5. Delete legends

  1. Open the Maintenance app and click Other > Legend.

  2. Click Legend.

  3. In the list, find the legend you want to delete.

  4. Click the options menu and select Delete.

  5. Click Confirm.

7.15.6. Display details of legends

  1. Open the Maintenance app and click Other > Legend.

  2. Click Legend.

  3. In the list, find the legend you want to view.

  4. Click the options menu and select Show details.

7.15.7. Assign a legend to indicator or dataelement

You can assign a legend set to an indicator or a data element in the Indicator/Data element module. This legend set will then be automatically selected when such an indicator/data element is selected in the GIS.
